# Task Summary

Create a complementary journal entry that features a mini comparison between two fruits, focusing on how they look, taste, and feel. Describe how each fruit might be used or enjoyed, and highlight a vivid contrasting detail (e.g., crisp skin versus succulent flesh, tropical aroma versus mild sweetness).

# Context

The goal is to practice comparing and packaging observations in a constructive tone. The journal entry supports the catalog from Assessment 1 by reinforcing note-taking habits and showing how stories shift when you highlight contrasts instead of similarities.

# Task Instructions

1. Pick two fruits, ideally different in colour, texture, and spirit. Add a short captioned photo/drawing for each.
2. Write a short paragraph for each fruit that covers its standout features and how it behaves when eaten, cooked, or served. Keep language upbeat and sensory-rich.
3. Write a closing paragraph that compares the two fruits and hypothesises how a chef or storyteller might use them together.
4. Save your journal entry as `journal.md`, and place the images in an `images/` directory alongside it.
5. Include a `readme.txt` that explains the inspiration sources and notes any third-party media used.

# Submission Instructions

Package `journal.md`, the `images/` directory, and the `readme.txt` into `assessment_02_output.zip`. Upload the ZIP to the shared workspace or learning management system.
